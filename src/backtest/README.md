# Package backtest

`backtest` 包提供了一个功能完备的策略回测框架，用于验证交易策略在历史数据上的表现。

## 核心功能

1. **Backtest 调度器:**
   - 模拟真实的交易流程。
   - 自动扣除佣金（Commission）。
   - 实时计算可用资金、持仓比例、总资产及最大回撤。

2. **多资产支持:**
   - 支持同时对多种资产进行并行回测。

3. **多样化报告:**
   - `DataReport`: 生成包含详细指标（如年化收益率、Sharpe 比率等）的内存对象。
   - `HtmlReport`: 自动导出精美的网页版报告，可视化展示回测结果。
