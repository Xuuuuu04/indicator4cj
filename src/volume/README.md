# Package volume

`volume` 包提供了结合成交量分析价格趋势的指标。

## 包含指标
- **成交量分布:** AD (Accumulation/Distribution), CMF (Chaikin Money Flow), MFI (Money Flow Index)。
- **成交量指标:** OBV (On-Balance Volume), NVI (Negative Volume Index), VPT (Volume Price Trend)。
- **价格加权:** VWAP (Volume Weighted Average Price)。
- **其他:** FI (Force Index), EMV (Ease of Movement)。
